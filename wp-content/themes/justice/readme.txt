== Description ==

Justice theme is responsive WordPress theme created for law companies, attorneys, legal offices, advocates, barristers at law, legal advisers. This theme is also useful for corporate, blogging, photography, consultant, restuarant, cafe and other industries. The modern, clean and professional layout makes sure your law firm builds trust right the minute a potential client visits your site. Justice theme is fully responsive work with every resolutions. This theme is compatible with the latest version of WordPress and with the plugins like woocommerce, contact form 7 and jetpack. Check demo at: http://flythemesdemo.net/justice/
